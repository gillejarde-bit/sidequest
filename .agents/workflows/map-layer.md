# Map Layer Workflow
Slash Command: `/map-layer`

## Steps
1. **Add Source**: Add the GeoJSON or vector tile `<Source>` to the MapLibre component.
2. **Add Layer**: Create the corresponding `<Layer>` (use `type="symbol"` for dynamic data).
3. **Add Interaction**: Add hover/click events, popup states, and cursor changes.
4. **Test Performance**: Verify performance with 100+ simulated features.
